# YrC Base Functions:
## How to set variable?
```
a = "1"
b = a
b += "1"
b -= "1"
b /= "1"
b *= "1"
```
## How to create final variable?(Only on YrC Native)
```
!a = "a"
!b = a
```
## How to join two strings?
```
a = "Count:"
b = "1"
strCat a and b to a
print a
```
## How to print?
```
print "123"
a = "123"
print a
```
## How to parse http(s)?
```
net_get_contents "https://google.com" a
print a
```
## What /SP - /RE?
/RE = "/n" (Return symbols)

/SP = " " (Space symbols)

# YrC Constructors
## Functions
### How to create custom function?
```
new Function myfunc
-print "This/SPis/SPmy/SPfunction!"
END.myfunc
```
### How to cast my function?
```
myfunc
```
